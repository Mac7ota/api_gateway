## TypeScript Client to make calls to an API Gateway built using AspNetCore.ApiGateway framework.

## Features of framework:

*	Swagger
*	Authorization
*   Filters
    *   Action
    *   Exception
    *   Result
*   Load balancing
*   Response caching
*   Web sockets
*   Event sourcing
*   Request aggregation
*   Middleware service
*   Logging
*   Clients available in
    *   .NET
    *   Typescript
